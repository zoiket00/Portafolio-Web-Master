Vertical 01 · Anime

Catálogo de anime en vivo, con datos reales.

Plataforma que consume la API pública de Jikan (MyAnimeList) para mostrar rankings en tiempo real. Incluye estados de carga (skeletons), manejo de error para límite de peticiones (429) con reintento, y filtros dinámicos por categoría (en emisión, próximos, populares, favoritos).
